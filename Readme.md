<h1 align="center">💰 Finance Dashboard UI</h1>

<p align="center">
A modern and interactive <b>Finance Dashboard</b> built using React.<br>
Helps users manage transactions, visualize financial data, and gain insights into spending behavior.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>Dashboard with summary cards (Income, Expense, Balance, Savings Rate)</li>
  <li>Time-based visualization (Monthly bar chart)</li>
  <li>Category-wise spending breakdown (Donut chart)</li>
  <li>Transaction management (Add, Edit, Delete)</li>
  <li>Search, Filter, and Sort transactions</li>
  <li>Role-based UI (Admin & Viewer)</li>
  <li>Dark mode support</li>
  <li>Export transactions as CSV</li>
  <li>Insights section (Savings rate, comparisons, trends)</li>
</ul>

<hr>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> React (Functional Components + Hooks)</li>
  <li><b>Styling:</b> Inline CSS (JS-based styling)</li>
  <li><b>Charts:</b> Custom-built using SVG & div</li>
  <li><b>Deployment:</b> Vercel</li>
</ul>

<h2>⚙️ Installation & Setup</h2>

<h3>1️⃣ Clone the repository</h3>
<pre><code>git clone https://github.com/Ravi-Saini01/Finance_Dashboard_UI.git
cd Finance_Dashboard_UI</code></pre>

<h3>2️⃣ Install dependencies</h3>
<pre><code>npm install</code></pre>

<h3>3️⃣ Run locally</h3>
<pre><code>npm run dev</code></pre>

<p>Open in browser: <b>http://localhost:5173</b></p>

<hr>

<h2>🏗️ Build for Production</h2>
<pre><code>npm run build</code></pre>

<p>This will generate a <b>dist/</b> folder.</p>

<hr>

<h2>🌐 Deployment (Vercel)</h2>
<ol>
  <li>Push project to GitHub</li>
  <li>Import project in Vercel</li>
  <li>Configure settings:</li>
</ol>

<pre>
Framework: Vite
Build Command: npm run build
Output Directory: dist
</pre>

<p>Click Deploy 🚀</p>

<hr>

<h2>🧠 Technical Decisions</h2>
<ul>
  <li>Used React Hooks instead of Redux for simplicity</li>
  <li>Built custom charts instead of using libraries</li>
  <li>Used mock data for faster development</li>
  <li>Implemented role-based UI for real-world simulation</li>
</ul>

<hr>

<h2>⚠️ Limitations</h2>
<ul>
  <li>No backend (data is not persistent)</li>
  <li>No authentication system</li>
  <li>Limited responsiveness</li>
  <li>No global state management</li>
</ul>
